Enterprise Group Policy Implementation in Active Directory

Company: Central Bank of Afghanistan

Duration: March 2016 – August 2021

As part of a strategic initiative to enhance security, standardization, and administrative efficiency across the Central Bank of Afghanistan’s IT infrastructure, I led the full design and implementation of a centralized Group Policy Object (GPO) architecture within the organization’s Active Directory environment.

The project began with a comprehensive assessment of existing policies and system behavior. I performed a full audit of legacy GPOs and organizational units (OUs), identifying conflicts, redundancies, and policy gaps. Based on this, I developed a new GPO structure aligned with Microsoft best practices and ISO/IEC 27001 compliance requirements.

Key implementations included:

  • Creation of tiered Organizational Units (OUs) for users, endpoints, and server roles
  • Design of security-focused GPOs for password policies, account lockout, and local group restrictions
  • Implementation of BitLocker drive encryption enforcement policies on all endpoints
  • Configuration of Windows Update policies using WSUS integration for centralized patch management
  • Control of removable media (USB, external drives) through GPO and registry lockdowns
  • Standardization of user desktop environments and taskbar configurations
  • Automation of GPO reporting and backup using PowerShell scripts

To ensure scalability and maintainability, I documented every GPO and its associated settings, created version-controlled backups, and implemented naming conventions and link order strategies. I also trained junior system administrators on managing GPOs and troubleshooting policy inheritance issues.

Technologies Used:
Windows Server 2012 R2 / 2016 · Active Directory · Group Policy Management Console (GPMC) · PowerShell · WSUS · BitLocker · Event Viewer

Outcome: The new GPO infrastructure significantly improved system security, reduced manual configuration efforts, and enabled consistent policy enforcement across 500+ client and server machines. It played a critical role in the bank’s successful compliance audits and long-term IT governance.
