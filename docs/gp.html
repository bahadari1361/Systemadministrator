🛡️ Active Directory Group Policy Implementation

📍 Central Bank of Afghanistan | 🗓️ 2016 – 2021

I led the full setup of Group Policies in Active Directory to improve security and standardize the configuration of 500+ systems. This included custom GPOs for USB blocking, BitLocker encryption, WSUS updates, and password policy enforcement.

🔧 What I Did:

  • Created tiered OU structure for users/devices
  • Designed custom GPOs for security and usability
  • Disabled USB ports and enforced BitLocker
  • Automated GPO backups and reports using PowerShell

🧰 Technologies Used:

Windows Server 2012 R2 / 2016, Active Directory, GPMC, PowerShell, WSUS, BitLocker

🚀 Impact:

Increased system security, reduced manual tasks by 40%, and ensured ISO 27001 compliance.
