Server Installations

Project: Windows Server 2022 Deployment

Scenario: A mid-sized company needed a new domain controller and file server to replace outdated servers and improve network performance.

My Role: I was responsible for planning, installing, and configuring the servers, ensuring minimal downtime during migration.

  • Tasks:
    • Installed and configured Windows Server 2022 on two physical machines.
    • Configured Active Directory, DNS, and DHCP services.
    • Set up file shares with appropriate user permissions.
    • Implemented regular backup schedules using Windows Server Backup.
    • Performed testing and troubleshooting for connectivity and access issues.
  • Technologies Used: Windows Server 2022, Active Directory, DNS, DHCP, Group Policy, Hyper-V for virtual testing
  • Outcome: Successfully deployed new servers with zero downtime. User authentication and file access improved, and IT staff now has a standardized backup process.
