Data Transmission Flow

  1. Sophos Firewall filters malicious DNS queries → Forwards clean traffic to Moltigo
  2. Moltigo DNS processes queries → Streams logs to:
    • Splunk (security events)
    • PRTG (performance metrics)
    • ntopng (traffic patterns)
  3. Wireshark (ad-hoc) captures packets for deep analysis
